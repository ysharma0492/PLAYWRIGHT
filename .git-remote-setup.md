# Git Remote Repository Setup Guide

## If you already have a remote repository:

1. **Add the remote repository:**
   ```bash
   git remote add origin <YOUR_REPO_URL>
   ```
   
   Examples:
   - HTTPS: `git remote add origin https://github.com/username/repo-name.git`
   - SSH: `git remote add origin git@github.com:username/repo-name.git`

2. **Push to remote repository:**
   ```bash
   git push -u origin master
   ```
   or if your default branch is `main`:
   ```bash
   git push -u origin main
   ```

## If you need to create a new repository:

### GitHub:
1. Go to https://github.com/new
2. Create a new repository (don't initialize with README, .gitignore, or license)
3. Copy the repository URL
4. Run the commands above

### GitLab:
1. Go to https://gitlab.com/projects/new
2. Create a new project
3. Copy the repository URL
4. Run the commands above

### Bitbucket:
1. Go to https://bitbucket.org/repo/create
2. Create a new repository
3. Copy the repository URL
4. Run the commands above

## Quick Commands:

```bash
# Add remote (replace with your URL)
git remote add origin <YOUR_REPO_URL>

# Push to remote
git push -u origin master

# Verify remote
git remote -v
```
